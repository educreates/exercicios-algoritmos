# Introdução - Introduction

*Modularização, Parâmetros Opcionais e Criação de Bibliotecas*

Os seguintes exercícios tem como objetivo praticar a construção de programas organizados em funções reutilizáveis, com foco em parâmetros opcionais, retornos múltiplos e separação do código em módulos.

Ao longo das atividades, foram desenvolvidoas pequenas funções em um arquivo separado, formando uma espécie de biblioteca própria de funções. Em seguida, essas funções serão importadas para outro arquivo, responsável pela lógica principal do sistema.

A proposta da lista é mostrar, na prática, como a modularização:
- reduz repetição de código;
- facilita a organização do programa;
- torna a solução mais legível e reutilizável;
- aproxima o desenvolvimento de exercícios da estrutura de pequenos projetos reais em Python.
- Em cada bloco, foi implementado um conjunto de funções que, ao final, foram utilizadas em um exercício integrador mais completo.

*Modularization, Optional Parameters, and Library Creation*

The following exercises aim to practice building programs organized into reusable functions, focusing on optional parameters, multiple return values, and separating code into modules.

Throughout the activities, small functions were developed in a separate file, forming a kind of personal function library. These functions are then imported into another file, responsible for the system's main logic.

The purpose of this list is to demonstrate, in practice, how modularization:

- reduces code repetition;
- makes program organization easier;
- makes solutions more readable and reusable;
- brings exercise development closer to the structure of small real-world Python projects.
- In each block, a set of functions was implemented and, at the end, used in a more complete integrative exercise.
